# React frontend for the MERN stack ecommerce website

### Method to setup in local system

1) Clone the repo for frontend
   ```shell
   git clone https://github.com/bikram-ghuku/ecommerce_client
   ```

2) Install Dependencies
   ```shell
   npm i
   ```

3) Run the web app
   ```shell
   npm run dev
   ```
